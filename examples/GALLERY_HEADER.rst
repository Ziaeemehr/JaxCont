JaxCont example gallery
=======================

These executable examples progress from classical equilibrium continuation to
the capabilities that distinguish JaxCont: batched whole-branch computation
with :func:`jax.vmap` and differentiable bifurcation locations. The gallery
also includes the standalone JIT-speedup demo.

The documentation build generates a downloadable Python script and Jupyter
notebook for every example. Read the Docs executes every gallery entry to show
its figures and captured output; local execution is opt-in. See
``docs/README.md``.
